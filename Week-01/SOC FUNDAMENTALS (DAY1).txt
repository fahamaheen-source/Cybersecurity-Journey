SOC FUNDAMENTALS
SOC (Security Operations Center)
A SOC (Security Operations Center) is a centralized unit consisting of people, processes, and security tools responsible for monitoring, detecting, analyzing, and responding to cybersecurity threats in real time.
It continuously protects an organization's systems from cyber attacks.
________________________________________
SOC PURPOSE
The main purpose of a SOC is to:
•	Protect company systems and data from cyber threats.
•	Detect security incidents as early as possible.
•	Respond quickly to attacks.
•	Reduce damage caused by cyber incidents.
•	Maintain continuous 24×7 security monitoring.
SOC = Detect + Respond + Protect
________________________________________
WHY ORGANIZATIONS NEED SOC
Organizations need SOC because:
•	Cyber attacks happen daily.
•	Attacks are becoming fast and automated.
•	Sensitive data is constantly targeted.
•	Early detection reduces damage.
•	Compliance requirements (ISO, GDPR, etc.).
•	Business continuity must be maintained.
•	Data breaches can cause financial loss and reputation damage.

SOC TIERS
A SOC is divided into three levels based on expertise and responsibilities.
________________________________________
TIER 1 SOC ANALYST (L1)
Entry-level analyst responsible for first-line monitoring.
Responsibilities
•	Monitor SIEM alerts
•	Validate alerts
•	Identify false positives
•	Perform basic investigations
•	Create incident tickets
•	Escalate serious threats to Tier 2
Focus
Monitoring and Alert Validation
________________________________________
TIER 2 SOC ANALYST (L2)
Intermediate-level analyst responsible for deeper investigation.
Responsibilities
•	Investigate incidents in detail
•	Analyze multiple data sources
•	Determine attack type
•	Perform incident response actions
•	Coordinate with IT and Security teams
Focus
Investigation and Incident Response
________________________________________
TIER 3 SOC ANALYST (L3)
Advanced security analyst responsible for threat hunting and security improvement.
Responsibilities
•	Threat Hunting
•	Malware Analysis
•	Detection Engineering
•	Create SIEM Detection Rules
•	Improve SOC Security Posture
Focus
Threat Hunting and Advanced Security Operations
________________________________________
SOC MANAGER
A SOC Manager supervises the entire SOC team and SOC operations.
Responsibilities
•	Manage Tier 1, Tier 2, and Tier 3 teams
•	Monitor SOC performance
•	Create security procedures
•	Coordinate with management
•	Ensure incidents are handled properly
•	Improve SOC efficiency
Goal
Ensure effective threat detection and incident response.
________________________________________
